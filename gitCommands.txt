git -> to see git installed , go to cmd and run "git"

git clone <clone repo
>
git add . -> add file changes in repository for staging

git add <file name>  ->adds one or more files to staging

git reset <file name> -> removes one or more files from staging

git status -> status of tracked and untracked files

git branch -> list available branches

git branch <branch name> -> create a new branch 

git checkout/switch <branch name> -> switches to different branch

git revert/reset -> <command>

git push -> push changes to centralized repo

Working Directory ---->Staging Area ----> Local Repository ----> Remote Repository

git fetch -> Remote Repository to Local Repository

git pull -> Remote repository to working directory
